# 坦克大战 — Tank Battle

经典 Battle City 风格坦克大战游戏，纯前端实现。

## 操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格 / J | 射击 |
| P | 暂停 |

## 游戏目标

- 消灭所有敌人坦克
- 保护基地（右下角鹰旗）不被摧毁
- 通过 3 个关卡

## 运行

直接用浏览器打开 `index.html` 即可开始游戏。

或者用 HTTP 服务器：

```bash
npx http-server . -p 3000
# 打开 http://localhost:3000
```

## 技术栈

- HTML5 Canvas
- 纯 JavaScript（无框架依赖）
- CSS3
