# 坦克大战 — Tank Battle

经典 Battle City 风格坦克大战游戏，纯前端实现。

## 操作

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 移动坦克 |
| 空格 / J | 射击 |
| P | 暂停 |
| G → O → D | 无敌模式（彩蛋） |

## 特色系统

| 系统 | 说明 |
|------|------|
| 🔥 **连击系统** | 连续击杀攒连击数，每 3 连击 ×1.5 倍加分 |
| 💥 **粒子爆炸** | 击杀/摧毁时粒子飞散效果 |
| ✨ **无敌闪烁** | 重生后短暂无敌，坦克闪烁提示 |
| 🏆 **高分记录** | 最高分自动保存在浏览器中 |
| 📊 **结算统计** | 游戏结束显示关卡、击杀、连击、最高分 |
| 🎬 **关卡转场** | 过关时显示 "STAGE X" 动画 |
| ☠️ **GOD 彩蛋** | 游戏中输入 G O D 开启/关闭无敌 |

## 游戏目标

- 消灭所有敌人坦克
- 保护基地（右下角鹰旗）不被摧毁
- 通过 3 个关卡

## 运行

直接用浏览器打开 `index.html` 即可开始游戏。

或者用 HTTP 服务器：

```bash
npx http-server . -p 3000
# 打开 http://localhost:3000
```

## 技术栈

- HTML5 Canvas
- 纯 JavaScript（无框架依赖）
- CSS3

## 开发

```bash
git clone https://github.com/myh1119/tank-battle.git
```
